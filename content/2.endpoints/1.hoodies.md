# Hoodies

## @GET All Hoodies
Endpoint to get all t-shirts from our T-shirts list.

```
method: get

/api/hoodies/all
```

## @GET Random Hoodie

Endpoint to get a random hoodie from our Hoodies list.

```
method: get

/api/hoodies/random
```

## @POST Find Hoodies

Endpoint to get a random hoodie from our Hoodies list.

```
method: post

/api/hoodies/find
```
